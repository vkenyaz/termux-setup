# Termux Setup

A simple shell script to set up my ideal termux environment on my Android
soy-phone. It installs packages and my personal termux shortcuts.

## Try it!

```
bash <( curl -fsSL https://codeberg.org/knyz/oracle/raw/branch/master/termuxsetup)
```
